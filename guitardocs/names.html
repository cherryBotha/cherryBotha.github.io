Guitar Basics: Names of strings and notes


Before getting into the notes, just a quick note, if you haven't already, you can watch the video this notes is based on, below this or on my YouTube channel. Guitar Cherry



  • The musical alphabet goes from A - G.
  • The String Names from the thickest to the thinest are - E, A, D, B, E.
    Remember This: Elephant And Donkeys Grow Big Ears.
  • Note Names from thinest string to thickest are - E, G, B, D, F
    Remember This: Every Good Boy Does Fine.
  • The Names of the Spaces from thinest string to thickest string are - F, A, C, E (NOTE: It spells FACE which rhyms with SPACE)
  • Each four fingers is assigned a finger

So putting all the names in order starting from the thinest string to thickest it would be:

  • G = string High E, third fret (high E the thinest string)
  • F = string High E, first fret
  • E = string High E, open
  • D = string B, third fret
  • C = string B, first fret
  • B = string B, open
  • A = string G, second fret
  • G = string G, open
  • F = string D, third fret
  • E = string D, second fret
  • D = string D, open
  • C = string A, third fret
  • B = string A, second fret
  • A = string A, open
  • G = string low E, third fret (low E the thickest string)
  • F = string low E, first fret
  • E = string low E, open

Music Notation

Ties add two note together to make one long note.

Dots add one half of whatever they are attached to.
